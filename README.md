# Self-Sustaining-solar-buoy-for-Oceanic-weather-and-pollution-analysis
## Overview
A solar-powered embedded system designed for real-time oceanic weather and pollution monitoring.

## Features
- Real-time weather monitoring
- Water pollution analysis
- Solar-powered operation
- Wireless communication using NRF24L01
- ESP32 and Arduino integration
- AI-based algae prediction

## Technologies Used
- Arduino
- ESP32
- NRF24L01
- Embedded C/C++
- Environmental Sensors

## Duration
August 2025 – March 2026
